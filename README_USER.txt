StreamBridge
============

Audio stream bridge for mAirList radio automation.

GETTING STARTED
  1. Launch StreamBridge from Start Menu or Desktop shortcut
  2. FFmpeg is included - no separate installation needed
  3. Default PCM stream endpoint: http://localhost:8765/stream
  4. API and PWA remote control: http://localhost:9000/app

MOBILE COMPANION
  Open http://<your-pc-ip>:9000/app on your phone
  Enable "Allow remote connections" in Settings > Remote

CONFIGURATION
  Settings are stored in: %APPDATA%\StreamBridge\config.json
